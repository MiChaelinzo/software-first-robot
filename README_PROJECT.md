## ðŸš¦ System Health Scoring

The simulation calculates a comprehensive 0-100 health score based on:
- Congestion levels (20 points)
- Robot utilization (15 points)
- Completion time efficiency (15 points)
- Collision avoidance (10 points)

**Score Ranges:**
- 80-100: Excellent performance
- 60-79: Good with minor improvements
- 40-59: Fair, optimization recommended
- Below 40: Poor, immediate action required
